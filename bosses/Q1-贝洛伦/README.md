## Boss 简介

贝洛伦是阿莱（Al'ar）最后一批孵化出的子嗣之一，被养育的目的是保护奎尔丹纳斯岛。在虚空尖塔事件之后，贝洛伦被虚空辐射污染。困惑且刚孵化的凤凰专注于它唯一的任务：阻止任何入侵者接近太阳之井。
