分团轮换进内场打盾，外场转火清小怪，白圈点名消水。

⚠️ 腐蚀毁灭
龙喷横扫场地，躲避直线区域

破盾后从内场出来，第一时间转火击杀，吃控制
🔶 吞噬瘴气

🔹 内场小怪
打破 20W 护盾，在蓝方块附近击杀
📋 P1 阶段
核心 分组轮换
  • 全团分两组，每组配 DK 拉怪
  • 一组场外，一组蓝方块分担进内场
  • 两组轮换，进内场后 90 秒不可再吃分担
内场 内场任务 (40秒)
  • 第一时间击破小怪盾
  • 在蓝方块附近击杀，让金水集中
  • 打完小怪转火 BOSS
外场 外场任务
  • 内场小怪破盾后出场走向 BOSS
  • 第一时间转火打掉
  • 小怪吃控制，T 根据大怪位置调整站位
点名 白圈消水
  • 被点名者离开人群，找金水最多处
  • 站定后奶妈驱散，不能秒驱
  • 驱散后炸掉金水清场
🔥 P2 阶段
核心 阶段转换
  • 两次进内场后转入 P2
  • BOSS 飞到台子上（无敌）
  • 满场都是水，召唤大量小怪
走位 大饼 → 三角
  • 大饼集合引导龙喷
  • 随后集体移动到三角
  • 龙喷留下直线污染和小怪，集合处理
📍 场地标记
标记 位置 用途
🔵蓝方块 场中央 分担进内场 / 内场小怪集合
🟠大饼 P2 起点 集合引导龙喷
🟢三角 P2 目标 引完喷吐后移动目标
⚪白圈 指定区域 点名消水站位
👥 职责分工
🛡️ 坦克
  • 将 Boss 坦克在边缘
  • 艾兰之尘时换坦
  • 拉住小怪在远离 Boss 位置
  • 根据大怪刷新位置调整站位
💚 治疗
  • 25秒/次 全团 AOE + 12秒后续掉血
  • 治疗压力极大
  • 白圈点名到位后再驱散
⚔️ 输出
  • 内场：快速破盾，蓝方块附近击杀
  • 外场：第一时间转火小怪
  • 小怪吃控制，DK 拉怪减速
🎯 开荒核心重点
轮换节奏
P1 两组进出场必须有序，内场打盾速度决定外场压力
消水管理
白圈点名必须精准覆盖金水，到位后驱散清场，不能乱放水
P2 位移
大饼 → 三角步调一致，避免被龙喷封死走位空间
