📋 机制要点
分担 晶体喷发
  • 大型棕色圈需要坦克去踩（分摊）
  • 环形波纹从落点向外扩散，跨过波纹进入安全区
  • 水晶墙由 BOSS 砸 T 地板后出现
转火 点名 虚空水晶
  • 在每堵墙上击杀 1 只 Blistereep 打破墙
  • 英雄模式：需要 2 只 Blistereep
  • Blistereep 会锁定随机玩家，需要风筝并击杀
核心 躲避 虚空吐息
  • BOSS 满能量大招，吐息横扫全场
  • 观察 Boss 确定吐息从哪一边开始
  • 摧毁两侧墙壁后，站到嘴两侧安全位置
  • 随吐息移动到场地的另一边
👥 职责分工
🛡️ 坦克
  • 一名坦克负责每个循环的前两次分摊
  • 之后换坦，直到下一个循环开始
  • 断桥机制：原地砸地板，T接伤害后易伤，一层一换
  • 马桶战：T 不要离开近战位
