开怪
双龙血量要接近 进盾躲好全屏波 点名出人调龙头 拉断连线散五码 踩球吸球别嫌累
💡 核心思路
双龙不共享血量，需保持血量接近（15%以内）同时击杀。满能量转阶段时进NPC护盾躲避全屏AOE。
🔄 转阶段机制
⚠️ 双龙飞天
满能量时双龙飞天，全屏AOE，必须进NPC护盾
💫 NPC护盾
20码内玩家获得30W吸收罩 + 抗减速
🐉 威厄高尔 (Vaelgor)
🔥 恐惧吐息
锥形喷吐，命中者恐惧15秒 + 高伤，可驱散
⚠️ 虚空光束/大黑圈
拉断连线，没人捆绑=团灭，H难度拉断后5码散开
💫 打主T
高额混伤 + 30%易伤debuff，一层换嘲
🐲 艾佐拉克 (Ezzorak)
🔥 虚空嚎叫
点名虚空圈，集合且分散放圈，转火打虚空球
🔥 大虚空球
踩球吸球减小体积，终点爆炸成黑水
💫 打副T
另一种混伤 + 30%易伤debuff，一层换嘲
📋 机制要点
核心 双龙机制
  • 血量必须保持在15%以内（英雄：10%）
  • 需要大约同时死亡
  • 坦克在相距15码的位置
  • 扫尾别朝中间
点名 恐惧吐息 (威厄高尔)
  • 随机点名，头前锥形喷吐
  • 被命中恐惧15秒 + 高伤掉血
  • 中点名出大团调整Boss头朝向
核心 虚空光束/大黑圈 (威厄高尔)
  • 头前激光后召唤大黑圈
  • 短暂延迟后捆绑附近玩家
  • 被捆绑者往外跑拉断连线
  • 没人捆绑 = 全屏高额AOE = 团灭
  • 英雄模式：拉断后5码散开
点名 转火 虚空嚎叫 (艾佐拉克)
  • 点名数个玩家虚空圈
  • 集合且分散，把圈放在一起
  • 出虚空球转火打掉（记得打断）
躲避 大虚空球 (艾佐拉克)
  • 头前吐息后召唤大虚空球
  • 球沿头前方向移动
  • 踩球吸球让球越来越小
  • 终点爆炸形成黑水（大小取决于球的大小）
👥 职责分工
🛡️ 坦克
  • 将两条龙坦克在至少15码外
  • 扫尾别朝中间
  • 虚空光束和幽暗机制面向中间
  • 恐惧吐息面向远离团队
  • 一层换嘲（debuff 30秒）
