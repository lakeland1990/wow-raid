## P1 阶段机制

### 威厄高尔 (Vaelgor) — 左侧 Boss

#### 龙喷 → 黑洞连线

1. BOSS 头前 AOE 吐息
2. 吐息结束，面朝方向召唤 **黑洞**
3. 黑洞 **牵引附近玩家**
4. 被牵引者需离开黑洞 **10 码** 拉断连线

> **灭团点**：每次拉断连线 → 全团 AOE 伤害

#### 换嘲时机

- 每次 **威龙 T 吃完龙喷后换嘲**

#### 恐惧吐息 (DREAD BREATH)

- **随机点名**一名玩家，扇形恐惧喷吐
- 被点名者 **穿过 BOSS**，朝向红叉(X)方向
- **不要喷到别人**
- 恐惧 15 秒 + 高伤 DOT，**可驱散**

---

### 艾佐拉克 (Ezzorak) — 右侧 Boss

#### 大虚空球 → 挡球机制

1. BOSS 头前 AOE 吐息
2. 召唤一颗 **大虚空球**，沿面朝方向移动
3. 大团 **分两组挡球**，每组最少 **5 人**
4. 按光柱标记顺序挡球：**星星 → 三角 → 紫菱**

> **踩球越小，终点黑水越小**

#### 虚空嚎叫 (VOID HOWL)

- 艾龙施放时，**全团 5 码分散**
- 脚下出虚空球
- 使用 **控制或打断技能** 解决

## P2 阶段：双龙飞天

> 满能量进入 P2 → 全团中间光盾集合 → 处理大怪 → 转回 P1

| 步骤 | 说明 |
|------|------|
| 1 | 全团在 **黄色大剑（中间光盾）** 集合 |
| 2 | 处理大怪 |
| 3 | 被点名的站 **圆圈边** |
| 4 | 坦克注意减伤，靠近拉你的龙 |
| 5 | 结束后转回 P1 |

| 职责 | 说明 |

| **威龙换嘲** | 每次吃完龙喷后换嘲 |

## DPS 职责

- **转火虚空球**：用控制或打断技能快速解决
- **分两组挡球**：每组最少 5 人，按标记顺序
- **拉断连线**：被黑洞牵引时往外跑 10 码
- **虚空嚎叫**：5 码分散
- **控制血量**：双龙血量差距保持在 10% 以内
