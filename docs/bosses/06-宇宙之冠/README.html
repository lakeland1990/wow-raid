💡 核心思路
多阶段战斗，核心机制围绕银锋箭消除玩家和小怪身上的虚空效果。P1 处理三只木桩大怪，P2 优先击杀增益型小怪，P3 是纯粹的DPS检查点。

📋 P1 - 不朽哨门
核心 银锋箭
  • 直线穿透，必须消层
  • 移除玩家和小怪身上的虚空效果
  • 是本场战斗最重要的交互机制
小怪 不朽哨门（三姊妹）
  • 三只木桩大怪，必须有T站位
  • 大Dài：周期性AOE，全团无法施法
  • 大Nì：周期性原地AOE，近战远离
  • 银锋箭可射掉虚空BUFF
躲避 空虚之握
  • 被拉后触发"崩裂空无"（射线/地板圈）
  • 第一时间逃离拉人中心
🔄 转阶段 (P1.5 / P2.5)
核心 银锋弹幕射击
  • 核心消层手段
  • 带有8秒300%增伤
  • 8秒后必须消一次，否则压力巨大
躲避 星辰散射
  • 击飞/击退效果随层数提高
  • 必须及时消层，防止被弹下平台
📋 P2 - 裂隙幻影
优先 裂隙幻影（黑奥）
  • 优先级极高！给Boss加减伤和伤害
  • "空虚之冕"吸收极高治疗 → 治疗爆发刷血
  • 必须打断"虚空弹幕"
小怪 不死的空灵爪牙
  • 必须打断"虚空弹幕"
  • 100能量后获得300%移速 + 500%伤害
  • 满能量前击杀或利用机制控制
换T 黑暗力量
  • 大怪和BOSS要拉开
  • 否则增伤加减伤，T属性降低（每层10%）
💀 P3 - 狂暴斩杀
DPS检查 噬灭宇宙
  • 场地逐渐缩减，吞掉即团灭
  • 全团99%减疗
  • 纯粹的DPS检查点
点名 终末箭
  • 点名需远离BOSS 30码以外
  • 被击飞空中，落地全团AOE
  • 吃羽毛加速跑
核心 重力坍缩
  • 被点名者需拉断连线
  • 未处理 → 300%物理易伤 → 基本倒坦
👥 职责分工
🛡️ 坦克
  • P1: 三只木桩大怪必须有T站位，无人即团灭
  • P2: 大怪和BOSS换T，否则属性降低（每层10%）
  • 注意大怪满能量自爆团灭
💚 治疗
  • 处理空无之冕（60W吸收盾）
  • 驱散会跳到其他人身上
  • 连续射击时注意加血
  • P3 全团99%减疗，资源紧张
⚔️ DPS
  • 优先击杀裂隙幻影（黑奥）
  • 打断不死的空灵爪牙"虚空弹幕"
  • P3 是DPS检查，全力爆发
