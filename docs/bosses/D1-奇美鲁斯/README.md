## 核心思路

**分团分组进内场打小怪护盾，外场转火清掉小怪。**

BOSS 场地周围的区域会召唤一大片大中小的小怪。团队需要分团分组，依次分组吃 BOSS 头前分担，被击中进内场的获得 40 秒击杀小怪的时间。小怪若不击杀，BOSS 满能量后会飞天吃掉所有小怪回血增伤。

**一句话：分组进内场把小怪打出来，外场的人减速杀，清掉所有小怪。**

## P1 阶段

### 艾兰之尘 upheaval (ALNDUST UPHEAVAL)

- BOSS 原地吼一嗓子，满场出现一大片大中小的小怪
- BOSS 打 T 的头前 AOE 分担圈
- **第 1 组** 应该分摊，这会将他们相位到另一个领域（内场）
- **内场限时 40 秒**
- 进内场后的队员 **90 秒不可再吃分担上台子**

### 内场任务

- 打断并打破所有护盾（小怪都有 20W 的盾）
- 将小怪送回现实（盾破掉后，小怪会来到外场）
- 内场玩家朝 BOSS 爬去

### 外场任务

- **第 2 组** 防止现实中的小怪到达 Boss
- 减速并击杀从内场出来的小怪
- 若被 BOSS 吃掉，它同样回血增伤

### 如果小怪没杀完

- BOSS 满能量会飞天，俯冲下地 AOE 全团
- 吃掉所有小怪给自己回血增伤
- 可能会周期性给全团上治疗吸收盾

### 吞噬瘴气 (CONSUMING MIASMA) - 英雄模式

- BOSS 周期性点名台子上的数个玩家魔法 dot 圈
- 驱散后，圈炸的伤害可以把地上的金水清掉
- 被驱散的玩家获得掉血 dot 外加击退
- 英雄模式下艾兰之尘水坑不会自动消失
- 随机玩家被施加吞噬瘴气，一个 DoT
- 当蓝色圆圈与艾兰之尘重叠时驱散玩家以移除水坑
- **一句话：酌情站好位置等驱散，别站在水过多的地方**

### 金水

- 小怪每次从内场出来，地上就会留下一摊金水
- 踩上去有伤害
- 驱散 dot 圈后可以消掉金水

---

## 过渡阶段

- 奇美鲁斯飞到空中，受到 99% 伤害减免
- 躲避横扫场地的腐蚀毁灭吐息
- 在过渡结束前击杀所有小怪
- **英雄模式**: 使用吞噬瘴气清除更多艾兰之尘

---

## P2 阶段

- BOSS 飞到台子上（无敌）
- 召唤一大片小怪
- 满场都是水

### 核心策略

- 大团转火小怪
- 尽可能在 BOSS 准备俯冲下地之前清干净小怪
- 否则回血增伤过高就很难受了
- 同样会点名数个玩家魔法 dot
- 驱散后会消水
- **水消的越多，身上 dot 越疼，击退距离可能越大**

---

## 坦克职责

- 将 Boss 坦克在边缘
- 当另一名坦克获得艾兰之尘 upheaval 时换坦
- 相位时拉住小怪，将它们坦克在远离 Boss 的位置

## 关键技能

| 技能名 | 描述 | 应对 |
|--------|------|------|
| 艾兰之尘 upheaval | 分坦分摊，进内场 | 两组轮流分摊 |
| 内场小怪 | 20W 护盾 | 打断护盾，击杀 |
| 外场小怪 | 从内场出来 | 减速并击杀 |
| 吞噬瘴气 | 英雄 DoT | 驱散清除水坑 |
| 金水 | 踩上有伤害 | 驱散消水 |
| 腐蚀毁灭 | 过渡吐息 | 躲避 |
| BOSS 飞天 | 吃小怪回血 | 尽快清小怪 |
