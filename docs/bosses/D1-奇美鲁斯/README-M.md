# 奇美鲁斯 - 史诗难度

> **副本**: 幽梦裂隙
> **难度**: 史诗 (Mythic)

---

## 史诗难度机制

### 内场 AOE

- 内场上打小怪的玩家，会周期性 **8 码自身 AOE 外场的玩家**
- 按规矩站位，不要故炸

### 迷失机制

- 内场的玩家，会有数个人迷失原地眩晕
- 迷失时，他们脚下会出现一个圈
- 外场的玩家，需要踩圈
- **交接棒进内外场**（外面的进去，里面的出来）

### P2 史诗难度

- 小怪也有内外场之分
- BOSS 会在飞天之前打一次分担圈
- 让半团人进内场，打 P2 内场小怪
- **内外场的两组人别互炸**

---

## 策略要点

1. 内外场玩家需要注意站位，避免互炸
2. 外场玩家需要及时踩圈交接棒
3. P2 阶段需要提前分好内外场组
4. 内场玩家迷失时，外场需要有玩家准备进去替换

---

## 与英雄难度区别

| 项目 | 英雄 | 史诗 |
|------|------|------|
| 内场 AOE | 无 | 8 码 AOE 外场 |
| 迷失机制 | 无 | 内场迷失，踩圈交接 |
| P2 小怪 | 仅外场 | 有内外场之分 |
| 分担进内场 | P1 | P1 + P2 |
