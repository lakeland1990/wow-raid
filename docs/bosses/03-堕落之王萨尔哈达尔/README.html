💡 核心思路
阻止 BOSS 吃黑洞，否则团灭。DPS 转火打掉黑洞。
⚡ 关键技能
⚠️ 虚空汇聚/黑洞
击杀球，吃掉即团灭，控制击杀速度
🔥 熵能解构
全屏激光躲避，Boss +25%受伤，开爆发
🔥 破碎暮光
点名出人群放尖刺，别指向Boss
💫 暴君命令
点名靠墙站放水坑
🔥 破碎投影
打断/控制所有分身，可瞬间击杀
🔥 召唤影子
迅速打断，读条成功会出紫水
📋 机制要点
核心 转火 虚空汇聚 / 黑洞
  • 两个球试图撞击 Boss，被吃即团灭
  • 在球撞击 Boss 前击杀它们
  • 英雄模式：错开死亡时间，避免叠加 DoT
  • 黑洞被打掉后，全团掉血 DOT，可叠加
躲避 爆发 熵能解构
  • 全屏 AOE 激光横扫，奶妈刷团血
  • 旋转激光束，大团躲避球
  • Boss +25% 额外伤害，持续 20 秒 → 开爆发
  • 能量清空后场上出现紫水，场地逐渐缩小
点名 破碎暮光 - 尖刺
  • 锁定坦克，向指示方向发射尖刺
  • 远离团队，避免将射线指向 Boss
  • 英雄模式：额外玩家也会被锁定
点名 暴君命令 - 放水坑
  • 点名数个玩家出人群散开
  • 靠墙站放水
  • 奶妈可驱散被点名者，驱散后刷掉吸收盾
打断 召唤影子
  • BOSS 召唤数个影子，必须打断
  • 每次成功读条都会召唤一大片紫水
👥 职责分工
🛡️ 坦克
  • 将 Boss 坦克在场地边缘
  • 将 Boss 风筝离开太靠近的球
  • 动荡打击层数高或破碎暮光时换坦
  • 随缘换嘲（叠层 DOT 而已）
⚔️ 全团
  • DPS 转火打掉黑洞
  • 躲避满能量时的激光
  • 被点名时靠墙站放水
