## 核心思路

1. **阻止 BOSS 吃黑洞** → 团灭技
2. **打断所有镜像** → 防止满地黑水
3. **点名出人群** → 榴莲和放水远离大团
4. **爆发留到旋转射线** → 20 秒易伤窗口

## 机制详解

### 虚空融合 - 黑洞 (重要)

场上共有 **三个装置**，BOSS 会在其中两个召唤 **凝结的虚空**（宝珠），缓缓飘向 BOSS。

| 机制 | 说明 |
|------|------|
| **宝珠碰 BOSS** | 团灭（虚空灌输：88W 伤害 + 1 分钟高额 DOT） |
| **玩家碰宝珠** | 虚空暴露：每秒 3W 暗影伤害 |
| **宝珠被打爆** | 晦暗侵蚀（英雄）：全团 8 秒高额 DOT，可叠加 |

**应对**：
- 远程 DPS 转火打掉宝珠
- **两个球错开击杀**，避免 DOT 叠加导致减员

---

### 粉碎暮光 - 榴莲点名 (英雄难度)

BOSS 点 **当前坦 + 3 个任意玩家**，5 秒后：
- 坦克吃到 67W 暗影伤害
- 四个人身上各射出 **5 道暮光尖峰**（榴莲）

**应对**：
- 被点名的 4 个人 **出人群**
- 不要把榴莲的尖角对着大团
- 尖峰踩上去每 2 秒 11W 伤害

---

### 破碎投影 - 分身 (打断)

BOSS 召唤 **5 个镜像**，位置随机。

| 行为 | 后果 |
|------|------|
| **镜像读条成功** (7秒) | AOE 全团 6W+ 伤害，留下一滩黑水 |
| **被打断** | 镜像直接消失 |

**应对**：
- 分配好打断链，每个镜像都需要打断
- 漏打断 = 满地黑水 = 场地越来越小

---

### 专制命令 - 放水 (治疗预警)

BOSS 点 **3 个玩家**，持续 10 秒：

| 阶段 | 效果 |
|------|------|
| **10 秒内** | 每 1 秒对 5 码内玩家造成 2.3W 伤害 |
| **结束后** | 压抑黑暗（11.7W 治疗吸收盾）+ 在脚下放一滩黑水 |

**应对**：
- 被点名的 3 个人 **跑出去贴墙放水**
- 治疗刷掉吸收盾
- ⚠️ 史诗难度不可驱散，只能硬吃 10 秒

---

### 扭曲遮蔽 (治疗预警)

全团环境伤害，无法规避：
- 直伤 3W + 23 秒 DOT（每秒 1.5W）

---

### 熵能瓦解 - 旋转射线 (重要/爆发窗口)

BOSS 能量满 (100) 时触发，持续 **20 秒**：

| 效果 | 说明 |
|------|------|
| **旋转射线** | 多道光束旋转，碰到每 0.3 秒 15W 伤害 |
| **全团 AOE** | 每秒 3.5W 伤害 |
| **BOSS 易伤** | +25% 受到伤害 |
| **结束后** | 原地放一坨巨大黑水 |

**时间轴**：
- 第一次：1:42
- 之后：每 2 分钟一次

**应对**：
- 嗜血、爆发药水、治疗大招都留到这里
- 坦克提前把 BOSS 拉离中场，避免污染后续场地
- 躲避旋转射线

---

## 坦克职责

**影蚀打击**：近战攻击叠加 DOT（每层每秒 8800 伤害，15 秒）

**换坦节奏**：
1. 出分身技能 → 换嘲
2. 出榴莲点名 → 换嘲

**其他**：
- 能量快满时把 BOSS 拉离中场
- 被粉碎暮光点名时出人群放榴莲
